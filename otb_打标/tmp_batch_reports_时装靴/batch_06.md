# 批次 6（图片 251–300）

### T01 鞋子类型
- 出现款式/颜色/材质及估算数量：基础短靴×15、马丁靴/机车靴×12、基础长靴×10、雪地靴×5、切尔西靴×4、西部靴×3、袜靴×1
- 未覆盖/难归类情况：无
- 边界模糊案例：
  - 图9、图22：TOD'S的短靴，鞋面有针织袜套，但鞋身主体是皮革，且有厚底，更偏向基础短靴的变种，而非纯粹的袜靴。此处归类为袜靴。
  - 图11、图12、图13、图17、图21、图26、图32：这些靴子虽然有搭扣装饰，但整体风格和厚底设计更符合马丁靴/机车靴的特征，而非基础短靴。

### T02 装饰物
- 出现款式/颜色/材质及估算数量：无×17、皮带扣×15、铆钉×6、品牌logo×6、金属扣饰×4、水晶钻饰×2
- 未覆盖/难归类情况：无
- 边界模糊案例：
  - 图4、图5、图16、图23、图27、图34：这些图片中的品牌logo是金属材质，且作为独立的装饰部件出现，因此归类为“品牌logo”。
  - 图31：白色切尔西靴，鞋身缠绕的水晶钻饰链条，虽然是链条形式，但其主要视觉效果是水晶钻饰，因此归类为“水晶钻饰”。

### T03 装饰位置
- 出现款式/颜色/材质及估算数量：无×17、鞋筒/脚踝×26、鞋侧×4、全鞋×2、鞋头×1
- 未覆盖/难归类情况：无
- 边界模糊案例：
  - 图31：水晶钻饰缠绕在鞋筒和脚踝处，且延伸至鞋身，但主要集中在脚踝和鞋筒，因此归类为“鞋筒/脚踝”。

### T04 图案装饰元素
- 出现款式/颜色/材质及估算数量：无×43、压花/印花/烫印×3、褶皱×2、动物纹×2
- 未覆盖/难归类情况：无
- 边界模糊案例：
  - 图1、图25：Maison Margiela的Tabi靴，鞋头有独特的“分趾”设计，这是一种结构上的设计，而非图案装饰元素，因此归类为“无”。
  - 图29、图30：Miu Miu的西部靴，鞋面有做旧效果，但没有明显的刺绣或雕花，因此归类为“无”。

### T05 颜色

- 边界模糊案例：

### T06 颜色类型

### T07 鞋面材质
- 出现款式/颜色/材质及估算数量：皮革/PU×36、麂皮/翻毛皮×9、毛绒/皮毛一体×3、针织/弹力布×1、漆皮×1

- 边界模糊案例：
  - 图9：TOD'S的短靴，鞋身主体是皮革，但鞋筒有针织袜套，由于针织部分面积较大且是其特色，因此归类为“针织/弹力布”。
  - 图32：黑色雪地靴，鞋面是漆皮，且有毛绒内衬，但漆皮是主要的外观材质，因此归类为“漆皮”。

### T08 鞋头形状
- 出现款式/颜色/材质及估算数量：圆头×23、尖头×16、方头×11

- 边界模糊案例：

  - 图20：TOD'S的短靴，鞋头略微收窄，但不够尖锐，介于圆头和尖头之间，此处归类为“尖头”。

- 边界模糊案例：
