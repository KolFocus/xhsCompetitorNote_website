# 质量保证（Quality Assurance）

## 目录定位

本目录用于沉淀“质量体系与策略”，聚焦标准、流程与度量，而非测试执行产物。

## 包含内容（建议）

- 质量目标与度量指标（Defect Density、通过率、逃逸率、阻断缺陷SLA等）
- 需求/设计/代码评审清单与准入准出标准
- 分支与发布质量闸门（构建、Lint、测试覆盖率阈值）
- 缺陷生命周期与优先级定义
- 回归与抽检策略
- 事故复盘模板与案例
- 与测试文档协作规范

## 与“04_测试文档”的区别

- `04_质量保证`：定义质量策略与标准（做什么、做到什么程度、如何度量）。
- `04_测试文档`：落地执行测试（怎么测、测了什么、结果如何）。

如需查看测试计划/用例/报告，请访问：

- `04_测试文档/README.md`
