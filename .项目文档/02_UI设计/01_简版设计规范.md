# 简版设计规范

## 1. 设计原则

### 1.1 核心原则
- **简洁明了**: 界面布局清晰，信息层次分明，减少不必要的装饰元素
- **一致性**: 统一的视觉风格和交互模式，标准化的组件使用
- **易用性**: 操作流程简单直观，减少用户学习成本，提供清晰的操作反馈

### 1.2 设计目标
- 提升用户体验
- 降低学习成本
- 提高操作效率
- 保证界面美观

---

## 2. 色彩规范

### 2.1 主色调
- **主色调**: #1890ff（antd 默认蓝色）
- **主色调悬停**: #40a9ff
- **主色调激活**: #096dd9

### 2.2 功能色
- **成功色**: #52c41a
- **警告色**: #faad14
- **危险色**: #ff4d4f
- **信息色**: #1890ff

### 2.3 中性色
- **背景色**: #fafafa（页面背景）
- **占位背景**: #f0f0f0（图片占位符、空状态背景）
- **边框色**: #d9d9d9
- **文字色**: #262626
- **次要文字**: #8c8c8c（灰色文字）
- **辅助文字**: #999（更浅的灰色，用于占位文字、空状态文字）
- **禁用文字**: #bfbfbf

### 2.4 使用说明
- 主色调用于主要操作按钮、链接、重要信息
- 功能色用于状态提示、警告、错误信息
- 中性色用于背景、边框、文字等常规元素

---

## 3. 字体规范

### 3.1 字体族
- **中文字体**: "PingFang SC", "Microsoft YaHei", sans-serif
- **英文字体**: "Helvetica Neue", Arial, sans-serif

### 3.2 字体大小
- **大标题**: 24px（页面主标题，如"全部笔记"）
- **中标题**: 16px（区块标题）
- **小标题**: 14px（卡片标题、表单标签）
- **正文**: 14px（常规文字）
- **辅助文字**: 12px（提示信息、说明文字、博主名称等次要信息）

### 3.3 字重
- **标题**: 600（加粗）或 500（中等加粗）
- **正文**: 400（常规）
- **辅助文字**: 400（常规）

### 3.4 行高
- **标题**: 1.5倍字体大小
- **正文**: 1.5倍字体大小
- **辅助文字**: 1.5倍字体大小

---

## 4. 间距规范

### 4.1 基础单位
- **基础单位**: 8px
- 所有间距应为 8px 的倍数

### 4.2 间距等级
- **小间距**: 4px（紧密元素间距）
- **中间距**: 8px（常规元素间距）
- **大间距**: 16px（组件间距）
- **超大间距**: 24px（区块间距、页面边距）

### 4.3 组件间距
- **页面边距**: 24px（上下左右）
- **组件间距**: 16px（卡片、表单等组件之间）
- **表单项间距**: 24px（表单字段之间）
- **按钮间距**: 8px（按钮组内部，使用 Space 组件）
- **Row gutter**: [16, 16]（栅格布局的水平和垂直间距）
- **卡片间距**: `marginBottom: 24px`（卡片之间的垂直间距）
- **过滤器标签**: `marginBottom: 8px`（标签与输入框之间的间距）

---

## 5. 组件规范

### 5.1 按钮
- **主要按钮**: 
  - 背景色：#1890ff
  - 文字颜色：#ffffff
  - 高度：32px
  - 圆角：4px
- **次要按钮**: 
  - 背景色：透明
  - 边框：1px solid #d9d9d9
  - 文字颜色：#262626
  - 高度：32px
- **危险按钮**: 
  - 背景色：#ff4d4f
  - 文字颜色：#ffffff
  - 高度：32px
- **图标按钮**: 使用 `icon` 属性添加图标（如 ReloadOutlined）
- **按钮组**: 使用 Space 组件包裹多个按钮，自动处理间距

### 5.2 表格
- **行高**: 64px（默认）
- **表头背景色**: #fafafa
- **悬停效果**: 背景色 #fafafa
- **选中效果**: 背景色 #e6f7ff
- **边框**: 1px solid #f0f0f0
- **横向滚动**: 当内容超出时使用 `scroll={{ x: 1200 }}` 或其他合适的宽度
- **排序提示**: 使用 `showSorterTooltip={false}` 隐藏默认排序提示
- **列宽设置**: 根据内容类型设置合适的列宽（如封面100px、标题300px、品牌120px等）

### 5.3 表单
- **输入框高度**: 32px
- **必填标识**: 红色星号（*）
- **校验提示**: 红色文字，显示在输入框下方

### 5.4 对话框
- **宽度**: 
  - 小对话框：400px
  - 中对话框：600px（默认）
  - 大对话框：800px
- **圆角**: 4px
- **遮罩**: rgba(0, 0, 0, 0.45)
- **内边距**: 24px

### 5.5 卡片
- **背景色**: #ffffff
- **圆角**: 4px
- **内边距**: 16px 或 24px
- **阴影**: 0 2px 8px rgba(0, 0, 0, 0.08)
- **间距**: 卡片之间使用 `marginBottom: 24px`，卡片内使用 `Row gutter={[16, 16]}` 进行栅格布局

### 5.6 分页
- **每页条数**: 10/20/50/100 可选（字符串数组：['10', '20', '50', '100']）
- **默认**: 每页 20 条
- **位置**: 居中显示
- **显示功能**: 
  - `showSizeChanger`: true（显示每页条数选择器）
  - `showQuickJumper`: true（显示快速跳转）
  - `showTotal`: 自定义显示格式（如："第 ${range[0]}-${range[1]} 条，共 ${total} 条"）

### 5.7 统计信息卡片
- **组件**: antd Statistic
- **布局**: Card + Statistic 组合，横向排列
- **颜色规范**:
  - 主要数据（如笔记总数）: #1890ff（主色调）
  - 成功数据（如品牌数量）: #52c41a（成功色）
  - 警告数据（如达人数量）: #faad14（警告色）
  - 危险数据（如缺失内容）: #ff4d4f（危险色）
- **图标**: 使用 prefix 属性添加图标（如 FileTextOutlined、TagsOutlined 等）
- **空值显示**: 使用 `?? '-'` 或类似方式显示占位符

### 5.8 标签（Tag）
- **组件**: antd Tag
- **颜色规范**:
  - 蓝色（blue）: 品牌标签、视频类型
  - 绿色（green）: 图文类型
  - 红色（red）: 广告标签
  - 橙色（orange）: 商业标签
- **图标**: 支持使用 icon 属性添加图标（如 VideoCameraOutlined、PictureOutlined）
- **可点击**: 品牌标签可添加 `cursor: 'pointer'` 和点击事件

### 5.9 头像（Avatar）
- **组件**: antd Avatar
- **尺寸**: 小尺寸使用 `size="small"`
- **占位符**: 显示昵称首字母
- **图片代理**: 外部图片需通过代理服务访问

### 5.10 图片（Image）
- **组件**: antd Image
- **占位符**: 
  - 背景色: #f0f0f0
  - 文字: "无图"（12px，颜色 #999）
  - 居中显示
- **样式**: 
  - `objectFit: 'cover'`（保持比例填充）
  - `borderRadius: 4px`（圆角）
- **代理**: 外部图片通过代理服务访问

### 5.11 下拉选择（Select）
- **组件**: antd Select
- **搜索功能**: 使用 `showSearch` 和自定义 `filterOption`
- **清除功能**: 使用 `allowClear` 允许清空选择
- **占位符**: 使用 `placeholder` 属性（如"选择品牌"、"选择博主"）
- **宽度**: 使用 `width: '100%'` 自适应容器宽度

### 5.12 日期范围选择（DatePicker.RangePicker）
- **组件**: antd DatePicker.RangePicker
- **格式**: `format="YYYY-MM-DD"`
- **宽度**: 使用 `width: '100%'` 自适应容器宽度

### 5.13 工具提示（Tooltip）
- **组件**: antd Tooltip
- **自定义样式**: 可添加自定义滚动条样式类（如 `.tooltip-scrollable`）
- **最大尺寸**: `maxWidth: 400, maxHeight: 300`
- **内容样式**: 
  - `whiteSpace: 'pre-wrap'`（保留换行）
  - `wordBreak: 'break-word'`（自动换行）

### 5.14 标签页（Tabs）
- **组件**: antd Tabs
- **标签高度**: 40px
- **激活标签**: 蓝色下划线（主色调）
- **未激活标签**: 灰色文字（次要文字色）
- **样式**: 遵循 antd 默认样式规范

### 5.15 搜索框（Input.Search）
- **组件**: antd Input.Search
- **宽度**: 根据使用场景设置（如 300px）
- **占位符**: 使用 `placeholder` 属性（如"搜索笔记标题、品牌..."）
- **图标**: SearchOutlined（内置）
- **功能**: 支持实时搜索

## 6. 布局规范

### 6.1 页面布局
- **顶部导航**: 60px（固定高度）
- **侧边栏**: 200px（展开状态）
- **主内容区**: 自适应宽度
- **底部**: 40px（可选）

### 6.2 内容布局
- **页面标题**: 24px（字体大小），加粗，`marginBottom: 24px`
- **面包屑**: 12px（字体大小），灰色
- **操作区域**: 右对齐
- **内容区域**: 左对齐
- **过滤器区域**: 使用 Card 包裹，`marginBottom: 24px`
- **统计卡片区域**: 使用 Row gutter={[16, 16]}，`marginBottom: 24px`
- **栅格布局**: 
  - 使用 Col 组件进行栅格布局
  - 过滤器标签：使用 `<strong>` 标签，`marginBottom: 8px`

## 7. 交互规范

### 7.1 状态反馈
- **加载状态**: 显示 loading 动画
- **成功状态**: 显示成功提示（绿色）
- **错误状态**: 显示错误提示（红色）
- **警告状态**: 显示警告提示（黄色）

### 7.2 操作反馈
- **按钮点击**: 视觉反馈（按下效果）
- **表单提交**: 显示提交状态
- **数据保存**: 显示成功提示

### 7.3 导航交互
- **菜单展开**: 平滑展开动画
- **页面切换**: 淡入淡出效果
- **标签切换**: 平滑切换动画

---

## 8. 动画规范

### 8.1 过渡动画
- **持续时间**: 0.3s
- **缓动函数**: ease-in-out
- **应用场景**: 页面切换、菜单展开、对话框显示

### 8.2 加载动画
- **旋转动画**: 1s linear infinite
- **脉冲动画**: 1.5s ease-in-out infinite
